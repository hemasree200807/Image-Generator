# 🖼️ AI Image Generator

A lightweight AI-powered image generation application built using **Python** and the **Pollinations AI API**. The application allows users to generate images from text prompts and automatically saves the generated image to the local system.

---

## 📖 Project Overview

This project enables users to create AI-generated images by simply entering a text prompt. The application sends the prompt to the Pollinations AI Image Generation API, retrieves the generated image, and saves it as **output.png**.

The project is simple, fast, and does not require downloading or running large AI models locally.

---

## ✨ Features

- 🖼️ Generate images from text prompts
- ⚡ Fast image generation using Pollinations AI API
- 🌐 No local AI model installation required
- 💾 Automatically saves generated images as **output.png**
- 🐍 Lightweight Python implementation
- 📦 Simple command-line interface

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/hemasree200807/AI-Image-Generator.git
```

### Navigate to the project folder

```bash
cd AI-Image-Generator
```

### Install the required packages

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

---

## ▶️ Usage

1. Run the application.
2. Enter a text prompt when prompted.
3. The application sends the prompt to the Pollinations AI API.
4. The generated image is downloaded automatically.
5. The image is saved as **output.png** in the project directory.

---

## 📌 Example

**Input Prompt**

```
A futuristic city at sunset with flying cars
```

**Output**

```
✅ Image created instantly: output.png
```

The generated image will be saved as:

```
output.png
```
